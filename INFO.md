# Guest PWA - Структура проекту

## Призначення проекту

Progressive Web App (PWA) для гостей готелів. Дозволяє гостям отримувати доступ до інформації про своє проживання через унікальний токен доступу, без необхідності реєстрації або авторизації.

## Основні функції

### Доступ по токену

- Унікальний лінк для кожного проживання
- Безпечний доступ без реєстрації
- Автоматичне визначення рівня доступу на основі статусу Stay

### Різні екрани залежно від статусу

- **booked** - базова інформація, контакти
- **occupied** - повний доступ, Wi-Fi дані
- **completed** - read-only інформація, відгуки
- **cancelled** - інформація про скасування

### PWA функції

- Можливість встановити на домашній екран
- Офлайн доступ до базової інформації
- Push-нотифікації (в майбутньому)

## Технологічний стек

- **Framework:** Vue 3 (Composition API)
- **Routing:** Vue Router
- **HTTP Client:** Axios або Fetch
- **Styling:** Tailwind CSS (або інший)
- **PWA:** Service Worker, Web App Manifest
- **TypeScript:** Строга типізація
- **Build Tool:** Vite

## Інтеграція з Backend

- **Base URL:** Налаштовується через env змінну
- **Endpoints:**
  - GET `/guest/access/:token` - отримання даних проживання
  - POST `/guest/access/:token/requests/cleaning` - запит на прибирання (майбутнє)
  - POST `/guest/access/:token/feedback` - залишити відгук (майбутнє)
